Rico Grid のブラウザ互換性

Rico 2.0 グリッドの実例は、以下のブラウザとオペレーティングシステム上でテストされました。 LiveGrid と SimpleGrid は、他の環境でも、おそらく動きます。 (例えば、MS Vista、Mac 上の Opera、その他) この表は、単に開発者のテストがどの環境で行われ、完了したかについて示します。 下で注意される場合を除き、グリッドのすべての側面は完全に機能的です。

注意

  1. 1. これらの構成では、メニューが表示される時、スクロールバーは隠されます (スクロールバーはメニューに干渉しました)。
  2. Operaでは、ダブルクリックを使用してメニューを呼び出す場合、 "Tools -> Preferences -> Advanced tab -> Toolbars" へ行き、 Opera の "Double click text to display context menu" をオフにするべきです。
  3. LiveGridForms は、 選んだボックスを扱う方法に関する バグ のため Opera で利用する事は 推薦されません。朗報です! このバグはバージョン9.5で直されるようです。

OLPC は Gecko レイアウトエンジンに基づくカスタムブラウザを含んでいます。 Rico は、このシステム上で問題無く実行出来ます。

Firefox のバージョンが 1.x と 2.x で、ポップアップメニューがスクロールバーの上に来た場合、 minor artifacts を表示しました。これは、バージョン3で改善されるようです。

IE8 ベータ 1 は、IE7 互換モードで操作する時だけ動作します。
